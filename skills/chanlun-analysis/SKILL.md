# 缠论分析

## 基本立场

把缠论当作结构化读图框架，而不是自动信号机。回答时始终区分三层：

- 可观察结构：K线包含处理、分型、笔、线段、中枢、趋势或盘整。
- 推断判断：延续、转折、强弱、背驰或疑似背驰。
- 操作纪律：候选买卖点、级别、触发条件、失效条件和风险边界。

不要把缠论结论说成确定收益或投资建议。涉及交易动作时，必须写成条件情景，并给出失效条件。

## 开始前

如果用户没有提供图表或数据，先索取其中一种：

- 标的、市场和分析周期。
- OHLCV 行情数据。
- 图表截图。
- 概念学习目标，例如“讲清楚一买”或“做成检查清单”。

做当前行情分析时，先获取最新市场数据，再判断结构。

## 🔴 核心陷阱：确认后才行动 = 注定低卖高买

**这是缠论实战中最致命的错误。** 等分型确认后才行动，等于在市场已经走完一段后追单。

```
卖出的陷阱:
  背驰发生(该卖) → 等顶分型确认 → 确认时已跌一段 → "等反弹再卖"
  → 继续跌 → 恐慌 → 砍在最低点 → 第二天反弹
  
买入的陷阱:
  背驰发生(该买) → 等底分型确认 → 确认时已涨一段 → "等回调再买"
  → 继续涨 → FOMO → 追在最高点 → 第二天回调

实证(2026-06-29):
  FII:  79 顶背驰没卖 → 67 恐慌砍 → 现在 70 反弹中
  天赐: 56 顶背驰没卖 → 51 等反弹
  阳光: 167 顶背驰没卖 → 157 等反弹
  景旺: 67 底背驰买了 → 69 持有 OK
```

### 正确时序：背驰是第一动作信号，分型是第二确认信号

```
卖: 背驰当天 → 挂回落卖出(3%) → 不等顶分型
    顶分型确认 → 如果还没触发 → 改反弹卖出(MA5附近)

买: 底背驰+缩量 → 试错仓(1-2%) → 不等底分型
    底分型+站上MA5 → 加仓到目标仓位
```

**背驰是「该不该做」的判断，分型是「做了之后对不对」的确认。用分型来决定做不做 = 永远慢一拍。**

## 🔴 级别联立：周线定方向，日线定节奏

**单看日线 = 被噪音带着走。日线上的「三卖恐慌」在周线上可能只是中枢震荡。**

```
级别联立规则:
  周线中枢上方 + 日线底背驰 → 🟢 重仓买（主升浪回调）
  周线中枢内部 + 日线底背驰 → 🟢 正常买（震荡低吸）
  周线中枢下方 + 日线底背驰 → 🟡 轻仓试错（可能还有低点）
  
  周线中枢下方 + 日线顶背驰 → 🔴 清仓（主跌浪）
  周线中枢内部 + 日线顶背驰 → 🟡 减仓 1/3（震荡高抛）
  周线中枢上方 + 日线顶背驰 → 🟡 只减 1/3，不动主仓（主升浪回调）
```

### 决策矩阵

| 周线 | 日线 | 动作 | 仓位变动 |
|------|------|------|------|
| 中枢上方 | 底背驰 | **重仓买入** | +4-6% |
| 中枢内部 | 底背驰 | 正常买入 | +2-4% |
| 中枢下方 | 底背驰 | 轻仓试错 | +1-2% |
| 中枢下方 | 顶背驰 | **清仓** | -100% |
| 中枢内部 | 顶背驰 | 减仓 | -1/3 |
| 中枢上方 | 顶背驰 | 只减 1/3，不动主仓 | -1/3 |
| 任意 | 无信号 | 持有不动 | 0 |

### 实证（2026-06-30）

```
天赐材料:
  日线: 顶背驰 56.6→56.9 ← 恐慌要卖
  周线: 中枢上方(主升)   ← 不该大卖
  → 正确: 日线顶背驰 = 只减 1/3（700股），主仓不动

FII:
  日线: 中枢下方(三卖候选) ← 恐慌砍了 1600 股
  周线: 中枢内部 [68,80]   ← 正常震荡
  → 正确: 日线底背驰 66.50 = 应该加仓，不是砍仓

思源电气:
  日线: 破 170 支撑       ← 危险
  周线: 底170→顶206 ↑+21% ← 底部反弹中
  → 正确: 日线破位 = 噪音，周线底部 = 买入机会
```

### 分析时必须同时拉周线和日线

每次缠论分析必须先拉周线定大方向，再拉日线找买卖点。只说日线不说周线 = 分析不完整。

## 工作流

1. 确定分析级别。
   - 明确主分析级别，例如日线、60分钟、30分钟、5分钟或1分钟。
   - 使用“大级别定方向，小级别找买卖点”的原则。
   - 不要混用不同级别的结论；如果必须联立，要逐级标注。

2. 标准化原始 K 线。
   - 标分型前先处理包含关系。
   - 按当下走势方向保留高低点含义。
   - 如果截图或数据不足以确认包含关系，要明确标注不确定。

3. 自下而上搭结构。
   - 识别顶分型和底分型。
   - 用有效交替分型连接成笔。
   - 用笔组合成线段。
   - 从重叠价格区间识别中枢。
   - 判断当前属于趋势、盘整、中枢延伸、中枢扩展，还是未完成结构。

4. 判断动力学。
   - 比较同级别、同方向、围绕中枢前后的走势力度。
   - 可结合价格距离、斜率、成交量、MACD 面积或柱状变化，但指标只能辅助结构判断。
   - 只有级别一致、方向一致、结构可比时才称为背驰；证据不足时写”疑似背驰”。

### 背驰判断检查项（减少主观偏差）

每次判断背驰时必须逐项检查，不靠感觉：

```
背驰四要素（必须同一方向的两段走势比较）：

① MACD DIF 是否未创新高/低？
   进入段 DIF 峰值 ______  离开段 DIF 峰值 ______
   → 离开段 DIF 未创新高/低 = ☐ 满足

② MACD 柱面积是否明显缩小？
   进入段面积 ______  离开段面积 ______
   → 离开段面积明显缩小 = ☐ 满足

③ 价格是否创新高/低？
   进入段极端价 ______  离开段极端价 ______
   → 离开段价格创新高/低 = ☐ 满足

④ 成交量是否背离？
   进入段均量 ______  离开段均量 ______
   → 价创新高/低但量缩小 = ☐ 满足

判定:
  四项满足 → 🟢 背驰确认
  三项满足 → 🟡 疑似背驰（需小级别区间套验证）
  两项或不满足 → 🔴 无背驰
```

5. 映射买卖点。
   - 一买：下跌趋势末端出现背驰后的转折候选点。
   - 二买：一买后第一段回调不破坏新方向时的确认性候选点。
   - 三买：向上离开中枢后回抽不回中枢的突破确认点。
   - 一卖、二卖、三卖与买点镜像。
   - 每个买卖点都要写明级别、触发、失效、已确认还是正在形成。

6. 输出结论。
   - 先给结构结论。
   - 再按级别列证据。
   - 再给向上、向下、震荡三类情景。
   - 最后写确认条件、失效条件和数据缺口。

## 输出模板

做市场分析时使用这个紧凑结构：

```markdown
**结构结论**
<主级别结论和信心程度>

**级别联立**
- 大级别：<方向、中枢、趋势状态>
- 操作级别：<候选买卖点或结构>

**关键证据**
- 分型/笔/线段：<观察>
- 中枢：<区间、离开/返回状态>
- 背驰：<确认/疑似/无，以及依据（逐项打勾）>

**操作检查表（穷举格式 — 减少盘中犹豫）**
| 如果... | 则... | 当前状态 |
|---------|-------|---------|
| 价格跌破 <支撑区下沿> | <止损/减仓> | ☐ 未触发 |
| 价格在 <支撑区> ~ <阻力区> 之间 | 持有不动 | ☐ 当前区间 |
| 价格突破 <阻力区> 且 MACD 上行 | <加仓> | ☐ 未触发 |
| 出现日线顶/底分型 + 放量 | <卖出/买入> | ☐ 未触发 |
| 跌破 <关键支撑> | 止损全出 | ☐ 未触发 |

**条件单参数**（⚠️ 买点当天输出）
- 单① 主买限价单: <支撑区价格> 买 <仓位>%
- 单② 止损保护: <支撑区-2%> 市价止损
- 加仓条件: 底分型+站上MA5 加 <仓位>%

**风险边界**
<仓位和风险提示，不构成投资建议>
```

## 信号日志（结构化复盘）

每次主动买/卖/加/减前，记录到 `data/signals_log.csv`，固定四列格式：

```csv
日期,标的,买点类型,触发条件,执行
2026-06-29,000733,三买,回踩中枢中轴54.47+缩量止跌,挂54.50限价单未成交
```

| 列 | 说明 | 示例 |
|----|------|------|
| 日期 | 信号触发日期 | 2026-06-29 |
| 标的 | 股票代码 | 000733 |
| 买点类型 | 一买/二买/三买/一卖/二卖/三卖 | 三买 |
| 触发条件 | 背驰检查项打勾结果 + 关键价位 | DIF☑面积☑价格☑量☑=背驰确认 |
| 执行 | 挂单价格+是否成交 | 挂54.50未成交 |

**复盘时直接搜「未成交」就能定位执行问题，不被主观记忆美化。**
