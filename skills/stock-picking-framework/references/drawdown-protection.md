# Drawdown Protection (主跌浪防护)

> 何时加载: 持仓 -10% 以上, 或大盘出现主跌浪信号时。
> 解决问题: "分批建仓 → 主跌浪接刀 → 止损出局 → 错过反弹" 的经典陷阱。
> 核心原则: 默认允许左侧抄底（抓 V 反弹）, 3 个切换信号触发右侧确认。

## 1. 主跌浪识别（4 项满足 2 项即认定）

```
信号 1: 跌破 60 日均线 (中期趋势破)
信号 2: 14 日 RSI < 35 (中期动能死)
信号 3: 大盘上证 / 创业板同步跌破各自 60 日均线
信号 4: 主线核心标的同步破位 (如 AI 算力 3 龙头同周内 -10%+)
─────────
认定主跌浪 → 该标的所有"加仓档"暂停 1 个月
解除条件: 反弹至 60 日均线之上 + 站稳 5 个交易日
```

## 2. 大盘状态过滤（每日盘前检查）

```
每日盘前 3 秒检查:
  上证 + 创业板 + 沪深 300 三大指数

🟢 健康市: 三大指数都在 60 日均线之上
  → 按正常仓位 / 倒金字塔执行

🟡 震荡市: 任一指数跌破 60 日均线
  → 加仓量减半, 试错档暂停建新, 现金 ≥ 15%

🔴 危险市: 三大指数都跌破 60 日均线 + 沪深 300 RSI(14) < 35
  → 暂停所有加仓, 现金 ≥ 30%
  → 仅减仓, 不建仓
```

## 3. 情景识别 — 左侧 vs 右侧切换

```
默认规则 (满足以下 3 项): 允许倒金字塔左侧抄底
  ✅ 主线硬 (无业绩 / 政策 / 龙头同跌信号)
  ✅ 大盘健康 (三大指数都在 60 日均线之上)
  ✅ 距 60 日均线 ≥ -15% (未深度破位)

3 个切换右侧确认的开关 (任一触发即切换):
  开关 1: 主线被证伪
    - 标的业绩公告负面 (扣非利润 -20%+)
    - 政策 / 主线方向反转
    - 同行业龙头 1 周内 -15%+

  开关 2: 大盘危险市 (防线 2 触发)

  开关 3: 个股深度破位
    - 距 60 日均线 < -15%
    - 14 日累跌 > -15%

右侧确认机制:
  - 起仓档照常执行 (试探仓 100 股)
  - 主仓档 / 底仓档必须等"右侧确认信号":
    1 根回踩不破前低的阳线 + 量比 > 1.2 + 站稳 24 小时
  - 破位档 (突破前期密集成交区) 才执行最后一档
─────────
原则: "确认底部" 而非 "猜底"
极端超卖 (RSI(6) < 20) + 主线硬 → 例外允许左侧 (抓 V 反)
```

## 4. 止损后冷静期

```
任何标的止损出局后:
  - T+0 至 T+14: 完全冷静期, 不能买回, 不管它涨多少
  - T+14 至 T+30: 评估期, 必须重新满足量化通过线 + 主线再确认
  - T+30 后: 解禁, 但需要重新研究主线和基本面
─────────
目的: 防止"清仓后看到涨, 心痒重新追入" 的反复止损陷阱
```

## 5. 单标的最大回撤上限

```
对每只持仓设置"成本回撤红线":
  核心档: 持仓成本 -20% 必须减半
  普通档: 持仓成本 -25% 必须减半
  试错档: 持仓成本 -15% 必须清仓
─────────
触发减半后:
  - 不再加仓
  - 等待主跌浪结束信号 (站回 60 日均线 + 站稳 5 日)
  - 如继续跌至 -30% (核心) / -35% (普通) / -25% (试错) → 全清
─────────
注: "持仓成本" 按建仓总成本计算, 不是当前价
```

## 实战案例 (2026-05-07)

```
案例 1: 隆基绿能 (主跌浪期)
  - 主线弱 (光伏过剩) ⚠️
  - 14 日跌 -5.81%, RSI(14) 22.9 → 触发主跌浪识别
  - 业绩持续亏损 → 开关 1 触发
  → 应用规则: 右侧确认, 不能左侧抄底
  → 当前 ¥16.53 不要看到极端超卖就加, 等反弹 ¥17 才减仓不是加仓

案例 2: 易点天下 (主升浪期)
  - 主线硬 (AI 应用) ✅
  - 大盘健康 ✅
  → 默认规则适用: 允许倒金字塔左侧加仓
  → 但如果跌到 ¥32 (距 60MA < -15%) → 切换右侧确认
```

```
原系统 (无主跌浪防护):
  期望收益 (单笔): +0.8%
  最大回撤估算: -25% ~ -30%

加入 5 道防线 + 情景识别:
  期望收益 (单笔): +6.6%
  最大回撤估算: -15% ~ -20%
─────────
关键交换:
  牺牲: 部分 V 形反弹的最低点 (情景 2/3 时切换右侧错过 -3% ~ -8%)
  得到: 主跌浪陷阱的避免 (-25% 到 -10%)
  净改善: 单笔期望 +5.8 个百分点
```
