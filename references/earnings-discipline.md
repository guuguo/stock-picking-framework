# Earnings Announcement Discipline (财报披露纪律)

> 何时加载: 持仓标的进入财报披露窗口 (T-7 至 T+10) 时。
> 解决问题: "看到业绩下降就清仓 → 利空出尽错过反弹" 的真实痛点。
> 真实痛苦: 汇川 ¥58 减半 → 现 ¥75.55 (+30%); 易点财报后反涨 +23% 至涨停。

## 三大铁律

### 铁律 1: 财报披露前后 3 天禁止任何操作

```
T-3 至 T+0: 不确定性最大, 多空分歧激烈, 不下单
T+1 至 T+3: 观察市场反应, 不下单
T+5 后:    根据反应模式做决策
─────────
违反代价: 在不确定性最大时操作 = 放大恐慌成本
```

### 铁律 2: 财报本身不是决策依据

```
错误: "Q1 业绩 -10% → 公司变差 → 减仓"
正确: "Q1 业绩 -10% +
      市场是否已 price in? +
      主线状态是否变化? +
      估值是否到位? +
      下一催化剂何时? = 综合判断"
─────────
财报是"已知信息", 已包含在当前股价中
真正的决策依据是"未知 → 已知"的过程是否改变了主线 / 估值 / 想象空间
```

### 铁律 3: "利空出尽" vs "业绩证伪" 必须区分

```
利空出尽: 主线硬 + 业绩短期承压 + 长期逻辑不变 → 加仓
业绩证伪: 主线弱 + 业绩持续恶化 + 长期逻辑动摇 → 减仓
─────────
判断标准: 主线状态 × (1 - 业绩证伪程度)
```

## T+5 决策矩阵

```
T+1 至 T+5 观察清单:

┌─ 价格反应 ─────────────────┐
│  跌幅 < 5% + 量能萎缩         │ → 利空出尽, 持有 / 加仓
│  跌幅 5-10% + 量能放大        │ → 部分证伪, 减半
│  跌幅 > 10% + 量能巨大        │ → 主线证伪, 清仓
│  反弹 + 量能放大              │ → 利空出尽确认, 加仓
└──────────────────────────┘

┌─ 业绩含金量判断 ─────────────┐
│  营收 + 净利同步增 → 真增长     │ → 加仓机会
│  营收增 + 短期利润降(烧钱)      │ → 多为利空出尽
│  营收降 + 利润降(主营萎缩)       │ → 多为真利空
│  营收降 + 利润增(降本)         │ → 利润率提升, 加仓
└──────────────────────────┘

┌─ 主线状态判断 ────────────────┐
│  主线硬: 政策 + 行业 + 龙头同向 │ → 反弹概率 75%
│  主线动摇: 行业承压 + 公司主线变 │ → 反弹有限
│  主线证伪: 行业崩 + 公司战略变  │ → 继续跌
└──────────────────────────┘
```

## 财报披露窗口操作流程

```
披露日历 (按月):
  1 月底: 业绩快报 (部分公司)
  4 月:  年报 + Q1 报
  8 月:  中报
  10 月: Q3 报

每只持仓在披露窗口:
  T-7: 跑量化扫描, 记录基线指标
  T-3 至 T+0: 暂停操作, 关注公告
  T+0 (披露日): 阅读财报 + 电话会议
  T+1 至 T+3: 观察市场反应, 不操作
  T+5: 根据决策矩阵做加 / 持 / 减判断
  T+10: 定期复盘, 看是否符合预期

特殊情况 (例外允许 T+1 操作):
  - 业绩远超预期 (≥ +50%): 可在 T+1 加仓
  - 业绩远低于预期 (≤ -50%): 可在 T+1 减半
```

## 真实案例库（必看）

```
案例 1: 汇川 (反面教材)
  披露前: 用户清仓一半, 假设价 ¥58
  披露后: 利空出尽 + 机器人主线切换 → 涨至 ¥75.55 (+30%)
  教训: "看到业绩下降就清仓"是错误模式
  正确做法: 应等 T+5 观察, 主线硬时持仓

案例 2: 易点 (反面教材, 2026/4)
  披露 (4/27): 研发 +90% + Q1 归母转亏 (利空)
  披露后 (4/28-5/7): 反涨 +23%, 5/7 涨停 ¥43
  原因: AI Agent 故事重启 + 主线硬 + 利空已 price in + AppLovin 美股映射
  教训: 财报披露当天 / 前 1-2 天的恐慌价不是合理价

案例 3: 隆基 (正面教材, 2026/5)
  披露 (4/28): Q1 -19.2 亿亏损扩大
  披露后: 价格继续下跌至 ¥16.55 (-5%)
  原因: 主线弱 (传统能源) + 业绩持续证伪
  教训: 主线弱时, 业绩利空是真利空, 反弹是减仓机会
```

## 个人行为反思

```
错误模式 1: "看到业绩下降就恐慌"
  本质: 用历史数据指导未来决策
  根源: 焦虑 + 不愿承受不确定性
  改正: 财报当天不操作, T+5 看反应

错误模式 2: "看到主线消息就追涨"
  本质: 错过启动期, 在情绪高点接盘
  根源: FOMO + 害怕错过
  改正: 主线消息后等回调, RSI < 60 才介入
```
