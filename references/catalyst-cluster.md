# Catalyst Cluster (催化集群分析)

> 何时加载: 行业出现重大新技术 / 政策 / 美股龙头业绩时, 或评估持仓中长期空间时。
> 核心洞察: "前瞻性" 不是看单点催化, 是看"催化集群"。

## 催化分级

```
单点催化 影响: +5-10% (估值修复)
集群催化 影响: +30-100% (估值重估)
生态催化 影响: 商业模式重塑 (PE 锚改变)
```

## 4 类催化

```
A. 行业催化集群: 多个新技术/产品同时发生
   例: AIGC 大模型集群 (Image-2 + Seedance + 豆包)

B. 政策催化集群: 政策叠加效应
   例: 反内卷 + BC 技术补贴 + 出清加速 (光伏)

C. 上下游联动: 产业链同步发酵
   例: 算力上游 (英伟达) → AI 应用下游 (易点)

D. 海外映射: 美股/港股领先 → A 股跟进
   例: AppLovin Q1 → 易点 / 智谱港股 → 三七
```

## 判断催化等级的 4 问

```
1. 是单点还是集群? (越多催化叠加越强)
2. 影响哪个财务指标? (毛利率/营收/费用)
3. 量级和时点? (Q2 还是 Q3, +5pp 还是 +20pp)
4. 是否已 price in? (市场反应 vs 真实空间)
```

## 案例 1: AIGC 集群催化 (易点 / 三七 / 乐鑫)

```
2026 H1 AIGC 大模型爆发时间线:
  2026/1     Grok Imagine 1.0       文生视频 (xAI)
  2026/2/12  Seedance 2.0          导演级 AI 视频 (字节)
  2026/4/22  GPT-Image-2           商用级图像 (OpenAI)
  2026/5/12  DALL-E 2/3 停服        强制升级 Image-2

对 AI 营销 (易点) 的"集群效应":
  单点 (GPT-Image-2): 素材降本 +5pp
  集群 (+ Seedance + 豆包): 全链路重塑 +20pp
  生态 (商业模式重塑): PE 锚从 50x → 30-40x

受益分级:
  直接 (AI 营销): 易点 / 蓝色光标 / 引力传媒
  间接 (AI 应用): 三七 (智谱) / 乐鑫 (豆包)
  基础设施: 工业富联 / 中际旭创 (算力上游)

前瞻判断: 中长期 +60-100% 空间, 但需 Q3 验证落地
```

## 案例 2: 光伏行业出清 (隆基)

```
催化:
  政策: "反内卷" 政策落地 (2025-2026)
  技术: BC 电池 量产爬坡 (2026-2027)
  供需: 中小厂商出清 (持续中)

受益分级:
  集群催化 → 但兑现时间长 (1-2 年)
  估值已部分反映 → 短期空间有限
```

## 案例 3: 美股映射"导火索" (易点 5/7 涨停)

```
时间线:
  5/6 美股盘后: AppLovin Q1 EPS $3.56 (+113%)
  5/7 A股开盘: 易点天下涨停 +20%

传导路径:
  AppLovin Q1 暴击 → 公允价值非线性修复
  → 易点 Q2 归母环比反转 +2.34 亿预期
  → 资金抢映射股, 涨停

教训:
  必须建立"美股映射标的财报日历"
  海外龙头业绩 → A 股映射股的"导火索"
  涨停往往是"昨夜事件 + 今早资金共振"
```

## 美股映射监控规则

```
对持仓中有"海外业务/代理/持股"的标的:
  □ 标记 1-3 个 美股映射龙头
  □ 跟踪美股龙头季报日期
  □ 财报披露后 24 小时内评估对 A 股标的影响
  □ A 股开盘前确认日盘策略 (加 / 持 / 减)
```

## 用法

```
1. 每周日跑一次"美股映射检查":
   - 下周有哪些映射龙头要披露财报?
   - 有哪些行业新催化产品发布?

2. 持仓标的进入"催化窗口"时:
   - 加载本文件评估催化等级
   - 不要错过集群叠加机会
   - 也不要被单点催化误导

3. 评估持仓估值时:
   - 单点催化: 用相对估值 (PEG)
   - 集群催化: 考虑估值重估
   - 生态催化: 调整 PE 锚
```
