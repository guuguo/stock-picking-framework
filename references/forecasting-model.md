# Forecasting Model (自有预估模型 FCEM)

> 何时加载: 计算战略档价位 / 任何 Y+2 净利预估 / 评估券商预期偏离。
> 核心洞察: **券商一致预期 ≠ 真实预期**, 自建四源交叉验证模型, 券商预期仅作 sanity check。
> 与 SKILL 联动: 铁律 19 (动态双锚定) 的价值锚必须基于 FCEM 计算。

---

## 一、为什么不能信券商一致预期

| 券商预期的常见问题 | 实际偏差 | 案例 |
|--------------------|---------|------|
| **乐观偏差** | 高估 15-30% | 买方付费 → 看多倾向 |
| **路径依赖** | 转折点失灵 | 抄上一轮预期, 每季微调 |
| **同质化** | "5 家平均" = 1 家结论 | 5 家券商抄 1 份原始研报 |
| **季节性更新** | 滞后 2-3 个月 | 财报后才调整 |
| **覆盖偏差** | 冷门股预期失真 | 热门 30+ 家 vs 冷门 0-3 家 |
| **不区分情景** | 无 Bear/Bull | 只给单一中性数 |

**实证案例 (兆易创新 2026/5)**:
```
券商一致预期 2026E 净利 = 20.1 亿
实际 2026 Q1 净利 = 14.61 亿 (Q1 一季就达到全年预期 73%)
偏离 = 严重滞后 (券商基于 2025 H1 数据)
```

**结论**: 券商预期 = **sanity check (健康度检查)**, 不是 **anchor (锚点)**。

## 二、FCEM 四源交叉验证模型

### 核心原则

```
4 种独立方法各算一次 Y+2 净利
↓
要求至少 3 个方法落在 ±25% 区间内 (一致性检验)
↓
取中位数作为 Base
↓
最低值 = Bear, 最高值 = Bull
↓
显式情景概率分配 (默认 25/50/25)
↓
期望加权
```

### 方法 1: 量价利 Bottom-up (业务线拆解) — V14.3 强化: 必须实证, 不能估算

```
Y+2 营收 = Σ (业务线 i 销量 × 单价)
Y+2 净利 = 营收 × 毛利率 × (1 - 期间费率) × (1 - 税率)

每个变量需 ≥ 1 个**实证**证据 (从券商研报/公告/财报抓取, 不能拍脑袋):
  销量:     公司公告的产能规划 / 在手订单 / 已锁单率
  单价:     行业报价数据 / 大客户合同披露 / ASP 趋势
  毛利率:   过去 8 季度均值 ± 产品结构变动 (财报附注)
  期间费率: 过去 3 年均值 ± 规模效应 (季度财报)
  税率:     公司有效税率 (财报)
```

#### V14.3 铁律: bottom-up 实证三档 (拒绝"估"字开头)

```
🚫 错误写法: "估算 2028 装机 ~600 MW, 单 MW 收入 ~3000 万"
   (这是从天而降的数字, 没有证据链)

✅ 正确写法:
   "2024 实证新增 IT 功率 +240 MW (华安证券 5/6 P3, 公司公告)"
   "2025 计划交付 IT 功率 ~420 MW, 全部锁单 (华安 5/6 P3)"
   "推演 2025 底总 IT 功率 ~1000 MW (240+420+前期 340 估)"
   "2025 营收 56.74 亿 / 假设有效功率 500 MW × 75% 利用率"
   "  = 1163 万/MW/年 (实证倒推, 非估算)"
   "2028 推演 IT 功率 ~1700 MW × 利用率 80% × 单 MW 1300 万"
   "  = 营收 ~177 亿"
```

#### bottom-up 4 个核心问题 (必答, 与 deep-research.md 维度 0 联动)

| 问题 | 推演变量 | 失败惩罚 |
|------|---------|---------|
| 当前装机 / 在建是多少? | Y+2 销量 | 不能出 Bull |
| 锁单率 / 订单可见度? | Y+2 销量置信度 | 散度 +10% |
| Top 3 客户占比 / Capex? | Y+2 销量天花板 | Bear 概率 +5% |
| 单 MW / 单订单 实际收入? | Y+2 单价 | 必须从财报倒推校验 |

**适用**: 业务结构清晰的标的 (兆易 / 工业富联 / 乐鑫 / 润泽 / 汇川等)
**不适用**: 早期成长股, 业务线不稳定 (此时方法 1 降权 + 用方法 4 同行映射)

**实证 1 (兆易 2028E, top-down 风险)**:
```
NOR: TAM 250 亿 × 市占率 30% = 75 亿
利基 DRAM: TAM 570 亿 × 12% = 68 亿
MCU: 中国 600 亿 × 9% = 54 亿
其他: 32 亿
合计收入: 229 亿
净利率 20% × 229 = 46 亿  ✓
```

**实证 2 (润泽 2028E, 真正的 bottom-up)**:
```
2024 实证: 新增 IT 功率 +240 MW (华安研报)
2025 锁定: 新增 IT 功率 +420 MW 全部已签订单 (华安研报)
推演 2025 底总功率: ~1000 MW (含历史积累)
2026-2027 假设增量: ~400 MW/年 (券商 EPS 隐含)
2028 底总功率推演: ~1700 MW

单 MW 经济 (实证倒推):
  2024 营收 43.65 亿 / 600 MW × 70% 利用率 = ~1040 万/MW
  AIDC 高密度溢价: 2028 升至 ~1300 万/MW

2028 营收推演: 1700 × 80% × 1300 万 = 177 亿
2028 净利率 (Q1 实证 31.6% + 规模效应): 40%
2028 净利: 177 × 40% = 71 亿  

⚠️ 注意: 此估算与券商一致预期 (~47 亿) 偏离 +51%
   → 必须解释偏离原因 (规模效应假设过激? 利用率过高?)
   → 修正: 取 method 1 与券商中位数加权 (60万 + 40)
```

### 方法 2: TAM × 市占率 Top-down

```
全球行业 TAM (Y+2) × 公司市占率 × 净利率
= Y+2 净利

每个变量需证据:
  行业 TAM:
    a. 龙头公司财报 (NVIDIA Capex 指引 / 台积电产能预测)
    b. 第三方 (Gartner / IDC / Yole / Frost & Sullivan)
    c. 头部行业协会 / 证券公司行业深度报告
    
  公司市占率:
    a. 过去 3 年市占率走势 (年报 / 第三方机构)
    b. 客户认证进度 (大客户列表)
    c. 产能扩张 / 技术领先性
    
  净利率:
    a. 同业中位
    b. 公司过去 5 年均值
    c. 规模效应 (营收翻倍 → 净利率提升 X%)
```

**适用**: 行业空间清晰 + 龙头标的
**不适用**: 全新赛道 (TAM 难估)

**实证 (兆易 NOR)**:
```
2028 全球 NOR Flash TAM = $35 亿 (IC Insights / 华经)
兆易市占率 25% → 30% (国产替代趋势)
= 250 亿 × 30% = 75 亿收入
```

### 方法 3: 历史增速回归 + 周期调整

```
过去 5-8 年净利 CAGR = X%
当前是周期上行 / 下行 / 中段:
  上行: 增速 = CAGR × 1.5
  中段: 增速 = CAGR × 1.0
  下行: 增速 = CAGR × 0.5

Y+2 净利 = Y0 净利 × (1 + 调整后增速)^2

适用: 周期成熟的赛道 (DRAM / 消费电子)
不适用: 全新赛道 (大模型应用) / 结构性变化期 (本轮 AI + 国产替代)
```

**⚠️ 实证警告 (兆易)**: 方法 3 计算出 32 亿, 但 AI + 国产替代 + 长鑫绑定 = 结构性变化, 本次研究剔除方法 3 结果。

**何时剔除方法 3**:
1. 行业经历技术革命 (AI 浪潮, 国产替代)
2. 公司经历重大战略转型 (并购/拆分)
3. 散度过大 (与其他方法差 > 30%)

### 方法 4: 同行 / 海外可比映射

```
找 3-5 家可比公司 (国内 + 海外)
计算它们: 现 PS / 现 PE / 历史增速

映射逻辑:
  公司 X 与可比 A 业务相似度 70%
  公司 X 处于扩张早期 (vs A 成熟期)
  → 公司 X 当前 PE 应该 = A 的 PE × 扩张溢价 1.5
  → 反推 Y+2 净利 = 公司 X 合理市值 / 当前 PE

适用: 有清晰对标的赛道 (HBM 对标美光 / 算力服务器对标 Dell)
```

#### V14.4 行业 PE 中枢分级表 (必查, 2026-05-26 新增)

> **2026-05-26 新增**: 案例驱动 — 易点 V1 用默认 32x PE 中位算 FCEM, 推出价值锚 ¥19, 但 V9 时给 PE 50x AI 主线溢价 (锚 ¥35-45). 默认 32x **系统性低估了 AI / 高成长龙头**, 必须按行业类型查表.

**取 PE 中枢前必先按行业类型查表, 不能默认 30-32x**:

| 行业类型 | 代表标的 | Bear PE | Base PE | Bull PE | 备注 |
|---|---|---:|---:|---:|---|
| **A. 成熟周期** (银行/钢铁/航运/光伏制造) | 工商银行 / 隆基 | 5-8x | 8-15x | 15-25x | 周期顶 PE 反低 |
| **B. 稳定成长** (消费/医疗/工控/电网设备) | 美的 / 平高 / 汇川 | 12-18x | 18-25x | 25-35x | PEG ~1 |
| **C. AI 高成长龙头** (半导体设计/IDC/光模块/AI 服务器) | 兆易 / 润泽 / 中际 / FII | 25-35x | 35-50x | 50-80x | 主线溢价 +30-50% |
| **D. AI 应用 / SaaS / 营销龙头** | 易点 / 三七 / 万兴 / 金山 | 25-35x | **40-50x** | 60-80x | AI 应用主线 PE 中枢 ≥ 40x |
| **E. 创新药 / 早期生物科技** | 百济 / 信达 | -- (亏损) | PS 8-15x | PS 20x+ | 用 PS 不用 PE |
| **F. ODM / 低毛利硬件** | FII (ODM 部分) / 闻泰 | 8-12x | 12-20x | 20-30x | 毛利低 PE 锚低 |
| **G. 电池 / 锂电材料 (周期反转)** | 天赐 / 恩捷 | 10-15x | 18-25x | 30-40x | 反转初期取 Bear, 兑现期取 Base |

**FCEM 方法 4 同行映射的正确顺序**:
1. 先按行业类型查表 → 锁定 PE 中枢区间 (Bear / Base / Bull)
2. 再找 3-5 家同行 → 取实际 PE 中位作 sanity check
3. 若同行 PE 中位 vs 表中 Base 偏离 > 30% → 说明行业整体处于 Bear 或 Bull 期, 不是中枢
4. 个股相对同行的溢价 / 折价 (业务相似度 / 龙头地位 / 国产替代) 在中枢上 ±10-20%

**禁止行为**:
- ❌ 不查表直接默认 32x 取中位 (易点 V1 错误)
- ❌ 用周期股 PE (10-15x) 给 AI 应用龙头 (应给 40-50x)
- ❌ 用 ODM 类 PE (15-20x) 给 AI 算力链龙头 (应给 35-50x)
- ❌ 看到 PE > 50x 就标"高估" — 类 C/D 高成长龙头 50x 是 Base, 不是泡沫

**实证 (易点 V1 错误)**:
```
易点 = D 类 (AI 应用 / 出海营销龙头)
正确中枢: Base 40-50x
V1 错误: 用 32x 中位 → 价值锚 ¥19 (系统性低估 60%)
V1.2 修正: 用 45x Base → 价值锚 ¥35.6 (与 V9 ¥35-45 一致)

兆易 = C 类 (半导体设计龙头)
正确中枢: Base 35-50x (Y+2 fwd)
若用 32x 默认 → 价值锚低估 30-50%
```

**实证 (润泽 2026/5/11)**:
```
润泽 = C 类 (AIDC 龙头)
表中 Base PE 35-50x
2028E fwd PE 33.7x = Base 区下沿合理 ✓
若误用 D 类 40-50x → 会低估
若误用 B 类 20-25x → 会严重低估 → 错误判定"高估"
```

#### V14.4 PE 中枢调整因子 (在表内中枢基础上微调)

```
+10% 因子 (累计最多 +30%):
  □ 行业绝对龙头 (市占率 > 30% 或 N1)
  □ 国产替代 / 自主可控核心环节
  □ 海外大客户独家代工 (NVDA / MSFT / META)
  □ A 股稀缺标的 (无对标)

-10% 因子 (累计最多 -30%):
  □ 客户集中度 > 50%
  □ 单一客户 > 30% 占比
  □ 业绩波动率 > 50% (单季 yoy 标准差)
  □ 历史 PE 震荡 (5 年区间 > 3 倍)

实证 (FII):
  D 类 ODM Base 12-20x
  + NVDA 独家代工 (+10%)
  + AI 算力主线核心 (+10%)
  + Q1 +94% 业绩硬 (+10%)
  → 调整后 Base PE 16-26x
  - NVDA 客户集中度 > 50% (-10%)
  → 最终 Base 14-24x → 取中位 18-22x ✓
```

**实证 (兆易 vs 华邦)**:
```
华邦 2028E 净利 28 亿 NTD
兆易业务相似度 85% (NOR + 利基 DRAM)
兆易溢价因子:
  - 国产替代 ×1.3
  - 长鑫绑定 ×1.15
  - A 股流动性 ×1.1
  → 总溢价 ×1.5-1.8
兆易 2028E 净利 = 28 × 1.5 = 42 亿 (Base)
```

## 三、一致性检验 (必做)

```
| 方法 | Bear | Base | Bull | 主要驱动 |
|------|-----:|-----:|-----:|---------|
| 1. 量价利 |  |  |  |  |
| 2. TAM × 市占率 |  |  |  |  |
| 3. 历史增速 ⚠️ |  |  |  |  |
| 4. 同行映射 |  |  |  |  |

Base 中位数 = median(B, E, H, K)
散度 = (max - min) / median
```

**散度处理规则**:

| 散度 | 含义 | 处理 |
|---|---|---|
| ≤ 20% | 高一致性 | 直接用中位数, 高置信度 |
| 20-30% | 中等 | 用中位数, 加大 Bear 概率 (30%) |
| 30-50% | 分歧大 | 标记 "不确定", 战略档不挂 (改挂深仓档代替) |
| > 50% | 方法失效 | 检查并剔除最不适用的方法, 重新检验 |

**剔除规则**:
- 结构性变化期: 剔除方法 3
- 新兴赛道 (TAM 模糊): 谨慎使用方法 2
- 业务多元化无明确对标: 降权方法 4

### 默认分配

```
| 情景 | 概率 | 假设 |
|------|-----:|------|
| Bear | 25% | 行业回落 / 大客户砍单 |
```

### 调整规则

| 触发条件 | 调整 |

| Bear 证伪信号 ≥ 2 个已出现 | Bear 概率 +10% |
| Bull 超预期信号 ≥ 2 个已出现 | Bull 概率 +10% |
| 散度 > 30% | Bear 概率 +5% (悲观偏置) |
| 黑天鹅事件 (L1) | Bear 概率 +20% |

## 六、合理 PE 选择

### 历史 PE 分析

```
取标的过去 5-10 年 PE 区间:
  最低 / 25% / 中位 / 75% / 最高
  剔除业绩异常年 (亏损 / 一次性收益)
```

### 同行 PE 对标

```
国内同行 PE 中位
海外可比 PE 中位 (调整 A 股溢价系数)
A 股板块 PE 中位 (流动性折溢价)
```

### Y+2 合理 PE 决定

```
情景 A: 周期回落 + 成长平台
  → PE 反映 "成长性可持续" → CAGR × 1.5 (PEG 1.0)
  
情景 B: 新一轮上行 (Bull)
  → PE 反映周期高点 → 取低值 (周期股原则)
  
情景 C: 周期低谷 (Bear)
  → PE 反映悲观预期 → 取高值 (低净利 × 高 PE 修正)

A 股溢价: ×1.2-1.5 (国产替代 + 流动性)
稀缺性溢价: ×1.1-1.3 (独家代工 / 龙头地位)
```

## 七、Forward PE 三年视角检验 (V14.3 新增, 必做)

> **痛点**: 看 PE(TTM) 经常误判 — 高成长股的 PE(TTM) 永远看似高 (兆易 75x, 润泽 68.7x), 但实际是定价未来
> **真实案例**: 润泽 V1 用 PE(TTM) 68.7x 判定"现价隐含 Bull, 全档期望负" → 用户质疑后才发现 Forward 2028 PE 仅 33.7x, 现价实际是定价 Base + 部分 Bull

### 必出 Forward PE 三年表

```
| 视角 | 净利来源 | 净利 (亿) | 隐含 PE | 估值评级 |
|------|---------|---------|--------|---------|
| TTM (滞后) | 当前 TTM | A | M/A | 参考 |
| Y+1 fwd | 券商一致预期 | B | M/B | **主决策** |
| Y+2 fwd | 券商一致预期 | C | M/C | 价值锚 |
| Y+3 fwd | 自有 FCEM | D | M/D | 长期空间 |

其中 M = 当前市值
```

### Forward PE 评级标准 (按行业类型)

```
A. 成熟周期股 (银行/钢铁/航运):
   Y+1 PE < 8x  低估
   8-15x       合理
   15-25x      高估
   > 25x       泡沫

B. 稳定成长股 (消费/医疗/工控):
   Y+1 PE < 20x  低估
   20-35x       合理
   35-50x       高估
   > 50x        泡沫

C. 高成长 + AI 主线 (半导体/IDC/光模块):
   Y+1 PE < 30x  低估
   30-50x       合理
   50-80x       略高估
   > 80x        高估 (但若 Y+2 PE 仍 < 40x 则可接受)

D. 算力服务器 / ODM (低毛利高 ROE):
   Y+1 PE < 15x  低估
   15-25x       合理
   25-35x       高估
```

### 三年视角的诊断逻辑

```
情景 1: TTM PE 高, Y+2 fwd PE 低
   例: 润泽 TTM 68.7x → Y+2 33.7x
   诊断: 现价定价了 Y+2 业绩兑现, 不算高估
   策略: 看 Y+2 业绩兑现路径 + Y+1 增速是否达标
   
情景 2: TTM PE 低, Y+2 fwd PE 也低
   例: FII TTM 28x → Y+2 18x
   诊断: 经典低估, 兑现度高
   策略: 重仓 (R11 6 选 5 通过 → 核心档)
   
情景 3: TTM PE 高, Y+2 fwd PE 仍高 (> 50x)
   例: 中际旭创 TTM 60x → Y+2 50x
   诊断: 现价已超额定价未来, 但若行业 alpha 持续可维持
   策略: 不重仓, 试错档观察 + 估值消化期不进
   
情景 4: TTM PE 低, Y+2 fwd PE 高
   例: 一次性收益拉高 TTM (剔除后 fwd 才真实)
   诊断: 业绩有水分, 不可持续
   策略: 不参与
```

## 七.5、战略档计算 (V4 - 修正"价值锚挂寂寞"问题)

**V3 问题**: 战略档 = 价值锚 × 0.8, 在主升浪期距现价 -40~-50%, 触发概率 < 10%, 等于不挂。

**V4 修正**: 把"价值锚"和"战略档"解耦, 分别承担"认知工具"和"实际挂单"两个角色。

```
认知锚 (不挂单, 仅作判断):
  价值锚 = Y+2 期望加权合理价 (FCEM)
  主线溢价上限 = 价值锚 × 1.5
  当前估值: 现价 / 价值锚 = +N%
    < +20%: 合理估值区
    +20% ~ +50%: 主线溢价区 (常态)
    > +50%: 主线高估区 (谨慎)
    > +80%: 严重高估 (战术档减半)
    
实际挂单 (战略档):
  战略档 = max(
      价值锚 × 0.85,    # 价值底线 (主升浪期取不到, 但合理期取得到)
      现价 × 0.80       # 现实底线 (普通回调可触发, 1-3 月内有概率)
  )

极端机会档 (可选):
  极端档 = 价值锚 × 0.8
  仅在以下情况挂单:
    - 极端档距现价 < -25% (太远的情况只挂极小量 1 手)
    - 黑天鹅事件 (L1) 后启用
```

### 战略档逻辑判断表

| 现价 vs 价值锚 | 战略档来源 | 距现价 | 触发概率 |
|----------------|-----------|--------|---------|
| < ×1.0 (低估) | 价值锚 × 0.85 | -15% ~ -25% | 高 |
| ×1.0 ~ ×1.2 (合理) | 价值锚 × 0.85 ≈ 现价 × 0.80 | -15% ~ -20% | 中高 |
| ×1.2 ~ ×1.5 (主线溢价) | 现价 × 0.80 | -20% | 中 |
| > ×1.5 (高估) | 现价 × 0.80 | -20% | 中低 |
| > ×1.8 (严重高估) | 现价 × 0.80 + 战术档减半 | -20% | 中低 |

### 实证 (兆易 2026/5/11)

```
价值锚 ¥222 (FCEM 加权)
现价 ¥362
比例 = 1.63 → 主线溢价区 / 接近高估
战略档:
  价值底 ¥222 × 0.85 = ¥189
  现实底 ¥362 × 0.80 = ¥290
  max = ¥290 ✓ (取现实底)
极端档:
  ¥222 × 0.8 = ¥178 (距现价 -51%, 仅挂 1 手作黑天鹅)
```

## 八、券商预期的正确用法 (sanity check)

```
我们算出 Y+2 Base = X 亿
券商一致预期 = Y 亿 (≥ 3 家平均, 排除 ≥ 6 个月前的旧预期)

if |X - Y| / Y < 30%:
    ✅ 自有模型与市场共识接近, 高置信度
    
elif X < Y × 0.7:
    ⚠️ 我们更悲观, 检查:
       - 是否券商乐观偏差?
       - 是否我们漏掉超预期信号?
    → 倾向用自有模型 (偏保守)
    
elif X > Y × 1.3:
    ⚠️ 我们更乐观, 检查:
       - 是否锚定效应 / 过度乐观?
       - 是否我们的 Bull 假设过激进?
    → 降为 X × 0.85 (向券商靠拢)
    
特殊情况: 季报后券商未更新 (滞后 > 3 个月)
    → 自有模型为准, 券商预期作废
```

## 十、模型局限性 (诚实声明)

```
1. 历史回归在结构性变化期失效 → 已通过"可剔除"机制处理
2. TAM 数据存在 ±20% 误差 → 用多家机构数据交叉
3. 同行映射假设业务相似度 70%+ → 相似度 < 60% 不可用
4. 净利率周期性强 → 必须明确"周期中段/顶/底"假设
5. PE 选择主观性 → 通过"历史分位 + 同行 + 海外对标"三维降低主观

最大盲点: 黑天鹅事件 (新技术颠覆 / 政策剧变)
  → 通过证伪信号 + 周度监控 + 持仓上限弥补
```
