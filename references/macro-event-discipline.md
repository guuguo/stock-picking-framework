# Macro Event Discipline (宏观事件纪律)

> 何时加载: 重要宏观事件窗口期 (T-3 至 T+5) / 海外大幅波动 / 政策语调变化 / 黑天鹅。
> 核心洞察: **大部分亏损不是选股错，是事件期硬扛或事件后追涨**。
> 与 `drawdown-protection.md` 联动: L1/L2 事件 + 大盘破 60MA = 启动主跌浪防线。

---

## 一、事件影响分级（决定操作幅度）

| 级别 | 描述 | 典型事件 | 操作纪律 |
|------|------|---------|---------|
| **L1 系统性** | 全市场休克，主线规则失效 | 重大地缘冲突 / 美股 -5%+ / 国内黑天鹅 | **暂停决策 3-5 天**，仅做止损 |
| **L2 结构性** | 板块大幅轮动，主线权重重估 | 政治局会议 / 中央经济工作会议 / 美联储重大转向 | T-3 至 T+5 暂停加仓，重评主线 |
| **L3 风格切换** | 成长/价值切换、风险偏好变化 | LPR/降准 / 美 CPI 大超预期 / FOMC 决议 | T-1 至 T+1 暂停起仓，已持仓不动 |
| **L4 情绪扰动** | 短期波动 ≤ 3% | 普通经济数据 / 行业政策 / 龙头业绩 | 不操作，不追不杀 |

**判断 L 级别的 3 个问题**：
1. 是否影响**全部**主线（L1）还是**部分**主线（L2）？
2. 是否引发**风险偏好**变化（L3）？
3. 是否仅是**短期情绪**（L4）？

## 二、固定事件日历（季度滚动跟踪）

> 🚨 **日历查证纪律 (R21 在宏观日历上的用法, 非新增规则)**
>
> **日历是"未经查证即默认错误"的数据**。固定日历会过时, daily_playbook 里的事件日期可能手误。
> 任何把"某事件在某日"当作行动依据 (尤其触发静默期/暂停加仓这类约束) 之前, **必须当场查证官方源**, 禁止凭记忆或直接抄旧表。
>
> ```
> 用日历做决策前必跑:
>   1. FOMC / 美联储事件 → 查 federalreserve.gov/monetarypolicy/fomccalendars.htm (或 web_search 交叉验证 ≥2 源)
>   2. 国内会议 (政治局/两会/中央经济工作) → 查官方公告, 不抄旧底稿
>   3. 财报日期 → 查交易所预约披露 (akshare/巨潮)
> 查不到确切日期 → 标 "待查", 不得据此触发任何"暂停/静默期"约束
> ```
>
> **错误的日历比没有日历更危险**: 它会凭空制造"今天该暂停"或"今天安全"的假约束, 直接误导当日操作。
>
> 实证锚点 (2026-06-03): daily_playbook 误录 "2026-06-04 FOMC", 致 AI 连续两次把 6/3 判为"FOMC 静默期暂停加仓"。
> 实际 2026/6 FOMC = **6/16-17** (官方源核实), 6/3 根本不在静默期。一个错日期 = 一个凭空的暂停约束。已更正。
>
> **2026 全年 FOMC (官方核实, 含点阵图者标 ⭐)**: 1/27-28 · 3/17-18⭐ · 4/28-29 · **6/16-17⭐** · 7/28-29 · 9/15-16⭐ · 10/27-28 · 12/8-9⭐

## 三、事件窗口期操作矩阵

### L2 事件标准窗口（FOMC / 政治局 / 中央经济工作）

```
T-7 至 T-4: 准备期
  ✅ 重评主线权重（事件可能改变政策风向）
  ✅ 检查持仓是否符合即将出现的政策方向
  ⚠️  减少新建仓（除非是 4 项准入 + 强信号）

T-3 至 T-1: 静默期
  ⚠️  起仓上限为试用仓(≤2%)，不建普通仓/核心仓
  ⚠️  已持仓加仓需"3 倍信号确认"（量价 + RSI + 主线）
  ✅ 减仓不受限（保护浮盈）

T+0: 事件日
  ⚠️  起仓上限为试用仓(≤2%)，不建普通仓/核心仓
  ✅ 盘后整理事件解读，但不做重大决策

T+1 至 T+3: 消化期
  ⚠️  等市场二次确认（避免假突破/假破位）
  ⚠️  起仓上限仍为试用仓(≤2%)
  ✅ T+3 后恢复正常仓位操作

T+4 至 T+5: 重定锚
  ✅ 根据事件后的新主线权重重评目标终态
  ✅ 调整观察价
```

### L3 事件简化窗口（LPR / CPI / 非农）

```
T-1 至 T+1: 起仓上限为试用仓(≤2%)，不建普通仓/核心仓；已持仓加仓正常
T+2 起: 正常操作
```

### L1 事件应急窗口（黑天鹅）

```
T+0: 立即扫描持仓
  ❌ 浮亏 > 5% 的非主线: 反弹即清
  ✅ 浮盈核心仓: 不动
  ✅ 现金 > 30% 时: 等明确止跌信号才入

T+1 至 T+3: 冷静期，仅止损不加仓
T+4 至 T+7: 等龙头主线企稳后再评估
```

## 四、政策语调判读模板

读政治局会议、中央经济工作会议、政府工作报告时，按以下结构提取：

```
1. 经济基调判断
   - 关键词: "稳中向好" / "面临挑战" / "外部冲击" / "防范风险"
   - 较上次会议: 升级 / 持平 / 降级

2. 财政政策力度
   - "积极的财政政策" + 几个修饰词
   - 赤字率 / 专项债额度（如有）
   
3. 货币政策力度
   - "稳健" / "适度宽松" / "灵活适度"
   - 是否提"降息降准"

4. 重点产业方向
   - 提到次数排序: 哪些主线被点名
   - 新词: 哪些是首次出现的提法
   - 删除: 上次提了这次没提的（往往是退潮主线）

5. 风险防范
   - 地产 / 地方债 / 金融
   - 直接影响: 银行、地产、城投平台

6. 对当前主线的映射
   - AI / 算力: 是否被强化
   - 半导体: 自主可控提法是否升级
   - 新能源: 是否被弱化（光伏锂电常见）
   - 消费: 内需提法力度

输出: 主线权重调整建议（升/降/持平）
```

## 五、海外联动判断（美股 → A 股）

### 美股下跌 vs A 股反应模式

| 美股表现 | A 股次日典型反应 | 操作 |
|---------|----------------|------|
| -1% ~ -2% | 低开 -0.5%，盘中修复 | 不操作，正常买卖 |
| -2% ~ -3% | 低开 -1%~-2%，分化 | 不加仓，观察主线 |
| -3% ~ -5% | 低开 -2%+，AI/科技重灾 | **暂停加仓**，等止跌 |
| > -5% | 低开 -3%+，全面回调 | **L1 应急**，仅止损 |

### 美债 / 美元 / 油价信号

```
美 10Y 收益率 > 4.5% 且持续 1 周:
  → 成长股估值压制
  → 减少 AI / 半导体加仓节奏

美元指数 DXY > 106:
  → 人民币贬值压力 → 出口股利好（消费电子/汽车出口）
  → 大宗商品压制（有色/煤炭）

WTI 原油 > $85:
  → 通胀压力 → 央行宽松空间收窄
  → 能源股利好

美股科技龙头业绩 NVIDIA / MSFT / META:
  → A 股映射 24 小时内: 算力 / AI 应用 / 光模块
  → 跟涨幅度 = 美股龙头单日涨幅 × 0.5-0.8（一般）
```

## 六、黑天鹅应急流程

### 触发信号（任一即启动）

- 单日大盘 -3% 以上
- 美股盘前/盘后 -3% 以上且确认延续
- 重大地缘冲突公告（战争/制裁名单/重大事故）
- 持仓中任一标的 -10% 黑天鹅

### 应急动作（按优先级）

```
Step 1: 现金弹药盘点
  - 当前现金 / 释放潜力（待止损/超目标仓位）
  - 计算"最坏情况下能补多少弹药"

Step 2: 持仓分级处置
  L1 持有: 浮盈 + 主线核心标的 → 不动
  L2 减半: 浮亏 0~5% + 主线但偏热 → 减半保护
  L3 止损: 浮亏 5%+ + 非主线 → 反弹即清
  L4 退出: 浮亏 10%+ + 主线证伪 → 立即清仓

Step 3: 冷静期
  T+0~T+3: 仅执行 Step 2，不加仓
  T+4~T+7: 等龙头止跌 → 主线重评
  T+8 起: 正常操作

Step 4: 复盘记录
  - 事件类型
  - 应急动作有效性
  - 是否违反纪律
  - 新增的"事件预警阈值"
```

```

  - 起仓上限: 仅试用仓(≤2%)，不建普通仓/核心仓

## 九、典型案例库（持续积累）

### Case 1: 2025 年 9 月 24 日 — 一揽子货币政策（L1 → L2）

```
事件: 央行降准 50bp + LPR 降 30bp + 房地产支持政策
T+0: A 股大盘 +4.15% (历史罕见)
T+1: 创业板 +5.4%, 半导体 +8%
应对: 已持仓不动, 现金 60% 状态下分批追入主线

教训:
  ✅ 事件级别准确判断 (L2 → 后续走出 6 个月主升浪)
  ❌ T+0 没追入 (太保守) → T+5 才进, 错过 +12%
  → 修正: L2 事件 T+0 收盘前 5% 仓位起仓核心标的
```

### Case 2: 2025 年 4 月 7 日 — 关税黑天鹅（L1）

```
事件: 美国宣布"对等关税"
T+0: A 股大盘 -7.5%, 创业板 -12%
应对: 当日暂停决策, T+3 后评估
  - 出口链 (家电/纺织) 减仓
  - 内需 / 自主可控 (半导体/AI) 维持

教训:
  ✅ 当日没有恐慌割肉
  ❌ T+1 早盘没补仓 (半导体 -15% 反而是机会)
  → 修正: L1 事件 T+1 早盘破位 -10% 时, 主线核心标的允许 1 手补仓
```

## 十、铁律

```
0. 用日历触发任何"暂停/静默期"约束前, 必须当场查证官方源 — 错日历 = 凭空的假约束, 比没日历更危险 (R21)
1. 事件窗口期，纪律 > 信号
2. 政策解读优先看"语调变化"，不要纠结字眼
3. 海外联动只决定"幅度"，不改变"方向"
4. 黑天鹅 T+0 禁止决策，T+3 才允许行动
5. 节前节后不加仓，避免假期黑天鹅
6. 美股暴跌时，A 股开盘半小时不操作
7. 国内重大会议 T-3 至 T+3 起仓上限为试用仓(≤2%)
8. 季度财报季 + 重要宏观事件叠加时取更严纪律
```
